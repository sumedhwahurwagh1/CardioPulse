# Screenshots & Demo Video Placeholders

This directory is designated for project screenshots, illustrations, and media demonstrating CardioPulse features.

## Recommended Screenshots to Capture

1. **`landing_page.png`**: Capture the hero section, key metrics layout, and ML feature cards.
2. **`diagnostic_wizard.png`**: Capture Step 3 of the patient assessment wizard showing clinical checkboxes and medical tooltips.
3. **`risk_results.png`**: Capture the results dashboard showing high-risk alert levels, radial SVG dials, and recommendation guidelines.
4. **`api_docs.png`**: Capture the OpenAPI documentation interactive page showing endpoints descriptions.

## Demo Video Guidelines

Capture a 1-2 minute walkthrough showing the form validation checks, network health indicator dot loading, and prediction workflow. Export as WebM/MP4 and host on an open CDN or commit to a storage branch. Link the video directly in the root `README.md`.
