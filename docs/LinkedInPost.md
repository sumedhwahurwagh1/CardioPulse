🚀 I’m excited to share my latest project: **CardioPulse**, a production-ready AI Heart Disease Classification System! 

CardioPulse bridges the gap between experimental data science and modern, secure software engineering. It takes medical diagnostic parameters (blood pressure, cholesterol, ECG, etc.) and predicts coronary artery disease risks instantly.

Here’s a breakdown of the technology stack and architecture:
* 🧠 **Machine Learning Core**: Built a modular Python pipeline comparing 5 candidate classifiers (Logistic Regression, Random Forest, SVM, KNN, Decision Tree) in Scikit-Learn, yielding **67.5% validation accuracy** and **73.4% ROC-AUC**.
* ⚡ **Backend REST API**: Implemented a FastAPI server featuring request model mapping (Pydantic), centralized exception middleware, and partitioned log outputs (API, training, errors).
* 🎨 **Frontend Client**: Designed a dashboard in React, TypeScript, and Vite, utilizing Tailwind CSS, Framer Motion, and Zod validations.
* 🐳 **Production Devops**: Dockerized the entire multi-stage stack (React/Nginx & Python Uvicorn) for single-command orchestration via Docker Compose. Configured GitHub Actions CI workflows for automated testing and compiles.

💡 **Key Takeaways**:
Moving ML models from a Jupyter notebook into production requires robust input validations, API error handling, and clean log partitioning. Building CardioPulse gave me deep hands-on experience in full-stack diagnostic engineering and MLOps.
